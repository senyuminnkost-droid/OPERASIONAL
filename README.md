# SENYUMINN KOST EXCLUSIVE PUNGGAWAN
## Sistem Manajemen Kost Eksklusif

Aplikasi manajemen operasional kost yang dirancang untuk efisiensi pengelolaan tenant, keuangan, dan pemeliharaan aset.

### Fitur Utama:
- **Dashboard Operasional:** Pantau status kamar dan keluhan secara real-time.
- **Manajemen Tenant:** Database penghuni lengkap dengan riwayat pembayaran.
- **Sistem Keuangan:** Pencatatan arus kas (Masuk/Keluar) yang transparan.
- **Manajemen Staff & Absensi:** Pantau kehadiran dan kinerja karyawan.
- **Weekly Service:** Jadwal pemeliharaan rutin kamar untuk menjaga kualitas aset.
- **AI Insights:** Analisis cerdas menggunakan Gemini AI untuk deteksi dini masalah operasional.

### Teknologi:
- React 19 (TypeScript)
- Tailwind CSS
- Gemini AI SDK
- Vite
